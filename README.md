# BLOG-WEBSITE

A full-stack blog website project with separate **frontend** (React) and **backend** (Node.js/Express) folders. This project allows users to read, create, and manage blogs with authentication support.

## Project Structure

BLOG-WEBSITE/
├─ frontend/        # React frontend
│  ├─ public/       # HTML, favicon, manifest
│  └─ src/          # React components, pages, styles
├─ backend/         # Node.js/Express backend
│  └─ ...           # Backend APIs and logic
├─ README.md        # Project documentation

## Features

- User authentication (Register / Login)
- Create, read, update, delete blog posts
- React-based responsive UI
- Backend APIs using Node.js and Express
- Clean project structure (frontend + backend)
